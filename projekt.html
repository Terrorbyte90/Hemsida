Ikarus
AI-agentplattform · Orchestrator/Worker

Avancerad AI-agentplattform med Orchestrator/Worker-arkitektur. Server-side Node.js-backend kör persistenta agenter som läser/skriver filer, hanterar git-operationer och rapporterar stegvis via WebSocket-streaming till en inbyggd iOS/macOS SwiftUI-klient. LLMCoordinator hanterar 50 req/min med max 6 simultana anrop — multi-provider routing (Claude, Grok, OpenRouter) med automatisk failover. Session-historik komprimeras automatiskt vid 72% kontextfönster för obrutna långa körningar.

AI-agent SwiftUI Node.js WebSocket GitHub API DigitalOcean Multi-provider
