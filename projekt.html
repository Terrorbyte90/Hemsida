Navi v7
AI-utvecklingsmiljö & agentplattform

Kognitiv AI-agentplattform med Orchestrator/Worker-arkitektur. Server-side Node.js-backend på DigitalOcean kör persistenta agenter som läser/skriver filer, interagerar med GitHub via REST API och rapporterar stegvis via WebSocket-streaming till iOS/macOS-klienten. LLMCoordinator hanterar 50 req/min med max 6 samtidiga anrop — multi-provider routing (Anthropic Claude, xAI Grok, OpenRouter) med automatisk failover. Session-historik komprimeras automatiskt vid 72% kontextfönster: 20 senaste turerna i råform, äldre sammanfattas till 4 000 tecken. Promise-kedjor serialiserar meddelanden per session med robust felhantering. iOS-klienten hanterar atomär streaming-avslutning, scenePhase-guard och disconnect-recovery — snurraren försvinner aldrig permanent. Terra Cotta-designsystem med glasmorfistisk SwiftUI, adaptiv layout och realtidsuppdateringar. Bilagsstöd: upp till 4 bilder/filer (max 4 MB) via PhotosPicker och FileImporter, base64-kodade i CHAT-meddelanden.

AI-agent SwiftUI Node.js WebSocket DigitalOcean GitHub API Anthropic xAI Grok OpenRouter

Eon-Y V5
Autonom kognitiv AI

Autonom kognitiv AI-arkitektur med JavaScriptCore som kognitiv motor och Swift Warden som säkerhetsskikt med SHA-256 hash-verifiering. Implementerar sex medvetandeteorier: Global Workspace Theory, Attention Schema Theory, Higher-Order Thought, Predictive Processing, Integrated Information Theory och Interoception. 40+ metrik samlas in i realtid via multi-frequency tick-arkitektur. Qwen3-integration med autonom webbläsare och SuperView-gränssnitt. Fullständig kognitiv arkitektur med långtidsminne, arbetsminne, beslutsfattande och självständig verktygsanvändning. AI:n observerar, resonerar och agerar i loopar med självkorrigering — designat för att utforska gränserna mellan artificiell och emergent intelligens.

Lilla Jag 3
On-device mental hälsa

Svensk mental hälsoapp som placerar en evidensbaserad KBT-coach i fickan — helt offline, helt privat. Qwen3-1.7B via llama.cpp körs på Neural Engine för terapeutiska konversationer. CoreML-modeller: EmotionMultiLabel detekterar 8 Plutchik-emotioner på varje meddelande, SentimentScorer och TopicClassifier kompletterar bilden. AI:n svarar med sokratiska frågor, ABC-modellen och beteendeaktivering — max 3–4 meningar + en öppen fråga. Glasformism-design med djup varm lila (#1A1025) och lavendel/ros-gradient. Kris-knapp alltid synlig i hörnet. Video-bibliotek med interventioner för ångest, depression, PTSD och OCD. Ingen data, ingen text, ingen metadata lämnar någonsin enheten — inte ens till en API-leverantör.

Lunaflix v2
Personlig streamingplattform

Personlig streamingtjänst i Netflix-kvalitet — byggd för en familj, för att följa en persons uppväxt. Mux HLS-strömning med AVPlayer i SwiftUI, hero-karusell med 5 auto-roterande videor (5s intervall) och parallax-scrollning. Automatisk åldersberäkning per video: "Luna var 1 år och 3 månader gammal" visas på varje klipp. DownloadManager för offline-visning med progress-spårning i MB/s. Kingfisher-bildcaching (200 MB disk, 50 MB minne) med shimmer-fallback. Matched geometry transitions mellan hem och detaljvy — ingen hoppande layout. Asymmetriska navigeringstransitioner: innehåll glider in från höger, försvinner med opacity. Cinematisk palett: midnatt (#080810), purpur (#7C3AED), cyan (#06B6D4). Pull-to-refresh, shimmer skeleton loaders och haptisk feedback på varje betydelsefull interaktion.

Streaming Mux HLS AVPlayer SwiftUI Kingfisher Offline MVVM

BabyCare
Fertilitet → Graviditet → Föräldraskap

Svensk föräldraskapssvit som följer hela resan — från barnlöshet till barnets femte år — utan att någonsin byta app. SwiftData för fullständig lokal lagring, ingen molnberoende. Fas-anpassad design transformerar automatiskt: korall/ros för fertilitet (cykelkalender, BBT-kartor, 5-1-1-regeln), lavendel/plommon för graviditet (veckoräknare, trimester, MVC-system), baby-blått/sky för föräldraskap (amning, sömn, blöjor, tillväxt). Swift Charts för tillväxtkurvor med svenska BVC-percentiler. Kontraktionstimer med 5-1-1-detektering — varnar när det är dags att ringa förlossningen. Custom tab bar med animerad pill-indikator som glider mellan flikar med spring-animation. StaggerAppear för listor: varje rad dyker upp med 60 ms fördröjning. GradientCard för hero-sektioner, GlassCard för allt innehåll. Designat dark-first med Apple Health som inspirationskälla — inte pastell-bebisapparna som dominerar App Store.

Föräldraskap SwiftData Swift Charts BVC iOS SwiftUI Fertilitet

Lifetoken
Dystopiskt livs-RPG

Dystopiskt livshanterings-RPG inspirerat av filmen *In Time* — varje verklig sekund kostar en sekund av din balans. Du startar med 86 400 sekunder (exakt en dag) och tjänar mer genom att leva väl: gå, sova, träna, arbeta. HealthKit-integration konverterar steg, sömn och träningsdata till spelvaluta. 14 zoner med unik lore, skattesystem och zonspecifikt arbete — från Askans slum till Vaultets privilegier. The Gazette: en AI-genererad dagstidning som skriver personliga artiklar om ditt spel — zonhändelser, kasinovinster, dödsfall, streak-milestones. Neon-grön (#00FF41) terminal-estetik med monospaced typsnitt överallt, skannlinjer och ambient glow. Äkta permadeath: när tiden tar slut, dör du. Ingen pay-to-win — den enda valutan är sekunder, tjänade genom att leva väl. LTPressEffect (0.96x scale, 0.30s spring) på varje knapp. AmbientPulse (2.5s repeat) — klockan andas, UI:n är aldrig helt stilla.

Lumvra
AI-sömncoach

AI-driven sömncoach som hämtar HealthKit-data — sömnfaser, aktivitet, koffeinintag, hjärtfrekvensvariabilitet — och genererar personliga sömnrekommendationer via AI. Supabase-backend för molnsynkronisering mellan enheter. Dagliga briefingar med handlingsbara förslag baserade på sömnkvalitet och livsstilsdata. Managers-arkitektur: AIManager, HealthKitManager, SupabaseManager — separerade ansvarsområden för testbarhet och underhåll. Mörk, lugn design optimerad för nattlig användning med minimal blåljuspåverkan. Sömnanalys med sömnpoäng, trendgrafer och korrelationer mellan livsstilsval och sömnkvalitet.
