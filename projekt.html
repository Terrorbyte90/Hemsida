AI-agentplattform

Kognitiv AI-agentplattform med Orchestrator/Worker-arkitektur. Server-side Node.js-backend kör persistenta agenter som läser/skriver filer, interagerar med GitHub via REST API och rapporterar stegvis via WebSocket-streaming till iOS/macOS-klienten. LLMCoordinator hanterar 50 req/min med max 6 samtidiga anrop — multi-provider routing (Claude, Grok, OpenRouter) med automatisk failover. Session-historik komprimeras vid 72% kontextfönster.
